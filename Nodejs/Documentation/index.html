Toner is a fully responsive and premium Nodejs ( EJS ) & Bootstrap 5.3.2 Ecommerce based Admin Template. Where you're creating a Dashboard, Products, Orders, Sellers, Shipments, Users etc pages which are required to create any ecommerce app then you are at the right place to buy Toner admin dashboard template. We have used Webpack-based build tools and SCSS variables-based modes along with RTL supported. You can quickly change the layout or mode by changing data attributes only instead of changing huge lines of codes. We have written complete minimal code for the beautiful UI. Toner has 5 types of layouts ( Vertical, Horizontal, Detached, Two Column & Hovered ) with multiple options of layouts like Fixed & Scrollable. Also support Dark, Light, and RTL modes and many other options.

Toner has 150+ pages, 40+ components, 30+ charts with fully responsive design in Desktop, Tablet and Mobile devices.
